Body Mass Index and daily calorie intake calculator

This app is used to calculate Body Mass Index of a person and daily calorie intake. User should enter his/her height and weight either in standard or metric units. Once the user has entered those values, he/she should click on compute BMI button. BMI table showing various categories and risk factors of being overweight is displayed on the side bar panel.

BMI Calculation

Once the use clicks on the calculate BMI button, the values entered by the user is displayed on the main panel and the BMI value and his/her corresponding BMI category are displayed on the main panel along with a contour plot of which BMI category the user belongs to. Formula for BMI Computation

      BMI = ( Weight in Pounds / ( Height in inches x Height in inches ) ) x 703
      BMI = ( Weight in Kilograms / ( Height in Meters x Height in Meters ) )

Formulas for BMI calculation and BMI categories are obtained from National Institute of Health.

Daily Calorie Intake Estimator

Once the use clicks on Daily Calorie Intake Estimator tab on the main panel, he/she is prompted to enter age, sex and type of their activity. Once all the information is entered by the user and the user clicks on Compute Daily calorie intake button

      calories required to maintain the current weight
      calories required to loose 1 lb/week
      calories required to loose 2 lb/week
      calories required to gain 1 lb/week
      calories required to gain 2 lb/week
are displayed. Mifflin - St Jeor equation to calculate BMR (Base Metabolic Rate) is
      For Men: 10*weightInKilograms + 6.25*heightInCentiMeter - 5*age + 5
      For Women: 10*weightInKilograms + 6.25*heightInCentiMeter - 5*age -161
Calorie requirements are computed by multiplying activity indicators (number from 1.2 to 1.9 based on the activity specified by the user) and BMR. To loose one pound per week 500 calories are subtracted from required calorie and to gain one pound per week 500 calories are added to the required calorie intake.
